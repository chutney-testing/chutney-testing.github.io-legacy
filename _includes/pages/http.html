HTTP tasks

HTTP Get

Presentation

The HTTP Get task performs a HTTP request with the GET request method

Inputs

Name Type - Format Mandatory Default value Validation
target string
uri string
headers map of strings
timeout duration string 2000 ms

Outputs

Name Type - Format
status int
body string
headers org.springframework.http.HttpHeaders (extends map)

Examples

Koltin
                                
HttpGetTask(
    target = "HTTP_TARGET",
    uri = "https://github.com/search?q=chutney",
    headers = mapOf(
    ),
)
YAML
                        
description: Github research
  implementation:
    type: http-get
    target: HTTP_TARGET
    inputs:
      uri: "https://github.com/search?q=chutney"
      headers: {}
      timeout: 2 sec
